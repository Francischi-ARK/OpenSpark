# OpenSpark 记账

手机可用的本地记账工具（PWA）。数据保存在浏览器 `localStorage`，无需登录。

## 功能

- 快速记一笔：支出 / 收入（大数字键盘）
- 记住上次分类；首页可一键复用最近几笔
- 分类、备注、日期
- 按月查看结余、收入、支出，以及分类占比
- 本月预算进度与超支提醒
- 点分类可筛选流水
- 编辑 / 删除流水
- 导出 / 导入 JSON、CSV 备份
- 识别 iPhone 截图 OCR 文字，自动填写微信 / 支付宝流水
- 通过快捷指令 URL 导入并按订单号去重
- 可安装到手机主屏，支持离线打开

## iPhone 一键截图记账

第一版不读取微信通知，也不上传截图。快捷指令使用 iPhone 自带 OCR，再把文字通过 URL 片段交给 OpenSpark；`#` 后的内容不会发送到 GitHub Pages 服务器。

在“快捷指令”中新建“截图记账”，依次添加：

1. **截屏**
2. **从图像中提取文本**，输入选择上一步的截图
3. **URL 编码**，输入选择提取的文本
4. **文本**，内容填写：
   `https://francischi-ark.github.io/OpenSpark/#import=编码后的文本`
   其中“编码后的文本”要插入第 3 步的蓝色变量，不要手动输入这几个字。
5. **打开 URL**

运行后 OpenSpark 会显示“确认识别结果”，核对金额、分类和日期后点“确认入账”。同一订单再次导入会提示重复。

可以把该快捷指令绑定到 iPhone **操作按钮**、**轻点背面**或主屏幕。微信和支付宝页面布局变化时，可在“设置与备份 → 截图识别”粘贴 OCR 文字进行排查。

## 手机怎么用

1. 合并本 PR 后，在仓库 **Settings → Pages** 开启 GitHub Pages（Source 选 `main` / root）。
2. 用手机浏览器打开 Pages 地址，例如：
   `https://francischi-ark.github.io/OpenSpark/`
3. 在浏览器菜单里选「添加到主屏幕」，即可像 App 一样使用。

本地预览：

```bash
python3 -m http.server 8080
```

然后访问 `http://localhost:8080`。
